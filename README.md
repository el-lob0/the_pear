# Commands

### .pear 
Main reason i made the bot. Does something funny to the message replied to with the command, if you put a gemini api key in .env

### .ask
Asks gemini ai something. Though i've put a limit on usage per user, cuz, well, theres a limit to my api calls lol.

### /encrypt 
Encrypt a message typed in the options

### !decrypt
Decrypt the message that was replied to with the command.
User decrypting must have his id in bot_storage/permission_to_decrypt.txt
